Flow-1: Pull Docker Image from Docker Hub and Run it
Step-1: Verify Docker version and also login to Docker Hub

docker version
docker login
Step-2: Pull Image from Docker Hub

docker pull vinramesh/springboot-cicdjenkensdockerintegratio
Step-3: Run the downloaded Docker Image & Access the Application
Copy the docker image name from Docker Hub

docker run --name app1 -p 80:8088 -d vinramesh/springboot-cicdjenkensdockerintegratio

// below command is working
docker run -p 9090:8088 <vinramesh: REPOSITORIES name>/<docker image> : springboot-cicdjenkensdockerintegratio>
docker run -p 9090:8088 vinramesh/springboot-cicdjenkensdockerintegratio

docker run -p 9090:8088 springboot-cicdjenkensdockerintegratio 
http://localhost:9090/test/welcome



http://localhost/hello
Step-4: List Running Containers

docker ps
docker ps -a
docker ps -a -q
Step-5: Connect to Container Terminal

docker exec -it <container-name> /bin/sh
Step-6: Container Stop, Start

docker stop <container-name>
docker start  <container-name>
